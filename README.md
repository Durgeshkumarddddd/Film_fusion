🎬 FilmFusion
FilmFusion is a dynamic and responsive web application designed for movie lovers to explore, search, and enjoy information about their favorite films. Whether you're looking for trending movies, classic hits, or upcoming releases, FilmFusion brings the magic of cinema right to your screen.

🚀 Features
🔍 Search for movies by title

🌟 View trending, top-rated, and upcoming movies

🎥 Get detailed movie info: description, cast, release date, ratings, and more

🎨 Beautiful and responsive UI/UX

📱 Mobile-friendly and optimized for all devices

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript, Bootstrap

Backend: Node.js, Express.js

Templating Engine: EJS

Database: MongoDB (if user data, watchlists, etc., are involved)

API: TMDb API for fetching movie data
